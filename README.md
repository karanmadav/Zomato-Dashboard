# Zomato-Dashboard
Data analysis and visualization of Zomato restaurant dataset using Tableau, PowerBI, MySQL, and Excel. Includes insights on cuisine trends, delivery ratings, and city-wise comparisons
# 🍽️ Zomato Dashboard - Data Analysis & Visualization

## 📌 Objective
Analyze Zomato restaurant data to uncover patterns in cuisine preferences, city-wise performance, delivery ratings, and cost trends across India.

## 💼 Tools & Technologies
- **Tableau** – Data visualization
- **Power BI** – Dashboard reporting
- **Excel** – Data cleaning & charts
- **MySQL** – Querying large datasets
- **PowerPoint** – Final project presentation

## 📁 Project Structure
| File Name                                  | Description                                      |
|--------------------------------------------|--------------------------------------------------|
| Excel Project - Zomato.xlsx                | Cleaned raw data, created charts & pivot tables |
| MYSQL Script - Zomato Project.sql          | SQL queries for filtering and aggregating data  |
| Power BI Project - Zomato.pbix             | Interactive visualizations in Power BI          |
| Tableau Project - Zomato.twbx              | Dashboard created in Tableau                    |
| ZOMATO P898 Restaurant Analytics.pptx      | Presentation summarizing insights               |

## 📊 Key Insights
- 🍕 **Top Cuisines** preferred across metro cities
- 🌆 **City-wise restaurant count & rating** comparisons
- 🚚 **Delivery performance** vs. Dine-in metrics
- 💰 **Average cost for two** across categories

## ▶️ How to Open Files
- `.sql` → Open in MySQL Workbench
- `.twbx` → Tableau Desktop or Public
- `.pbix` → Power BI Desktop
- `.xlsx` → MS Excel or Google Sheets
- `.pptx` → MS PowerPoint
